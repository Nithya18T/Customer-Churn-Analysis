# Customer Churn Analysis

## Project Overview
Customer Churn Analysis is a data analytics project that analyzes customer subscription behavior to identify the main reasons behind customer cancellations (churn).

The project evaluates customer engagement, tenure, monthly charges, and contract types to discover behavioral trends and factors affecting customer retention.

The analysis also provides business recommendations to improve customer retention strategies.

---

## Objectives
- Analyze customer behavior data
- Identify churn patterns and trends
- Compare churned and retained customers
- Visualize important business insights
- Suggest retention improvement strategies

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Dataset
The dataset contains customer information such as:
- Gender
- Senior Citizen Status
- Partner and Dependents
- Subscription Tenure
- Internet Service Type
- Contract Type
- Monthly Charges
- Total Charges
- Churn Status

---

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Churn Distribution Visualization
- Contract Type Analysis
- Monthly Charges Analysis
- Tenure Distribution Analysis
- Correlation Heatmap
- Business Insights and Recommendations

---

## Installation

Install the required libraries using:

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python churn_analysis.py
```

---

## Output
The project generates:
- Customer churn distribution graph
- Contract type vs churn analysis
- Monthly charges analysis
- Tenure distribution graph
- Correlation heatmap

All generated graphs are saved inside the `images/` folder.

---

## Key Insights
- Customers with month-to-month contracts churn more frequently.
- Higher monthly charges increase churn probability.
- Customers with shorter tenure are more likely to cancel subscriptions.
- Long-term contracts improve customer retention.

---

## Retention Recommendations
- Offer discounts for long-term contracts
- Improve customer onboarding experience
- Provide loyalty rewards and offers
- Increase customer engagement
- Introduce personalized retention strategies

This project helps businesses understand customer behavior and identify factors leading to customer churn. The insights generated can support better decision-making and improve customer retention strategies.
